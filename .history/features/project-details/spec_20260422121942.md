# Feature: Project Details Page

## Goal

Implement a page that shows detailed information about a selected project.

## Designs

![target layout](/assets/images/projectdetails.jpeg)

## Work

- The page will display the project title and description
- Include details about how the project works
- Possibly include images or screenshots
- Layout will be simple and easy to read
- Navigation back to the home page

## Deliverables

- A working details page
- Information is clearly displayed
- User can return to the home page
